# Archive (legacy / non-canonical)

Files in `docs/report/archive/` are **historical artifacts** (older briefs, dense drafts, and planning notes). They may contain **stale numbers or superseded framing**.

## What to trust instead

Use these as the project’s current “source of truth”:

- `results/REPORT.md` (canonical verified narrative)
- `results/EXECUTIVE_SUMMARY.md` (canonical short layer)
- `results/*.json` (metric source of truth)
- `results/final_bundle/*` (submission-ready bundle)

If any claim in the archive disagrees with the above, treat the archive as **outdated**.
